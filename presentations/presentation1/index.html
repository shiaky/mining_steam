What is Steam?

  • Digital distribution platform (developed by Valve Corporation)
  • Initial release: September 12, 2003 (14 years ago)
  • Largest digital distribution platform for PC gaming
  • By the end of 2015, the service had over 125 million registered accounts, with 12.5 million concurrent users

Information richness

  • What information are accessible?
  • Which interesting information aren't available?
  • Which information are protected/hidden?

Accessible Information

  • Games owned
  • Playtime (last 14 days and total)
  • Achievements owned
  • Badges owned
  • Account age
  • Real name [optional]
  • Location (City, Country) [optional]
  • ...

Unavailable Information

  • Age
  • Gender
  • Money spent on steam (in total)

Hidden/Protected Information

  • The user can set his profile state to private
  • Then, no data can be extracted üôÅ
  • But by default, all profiles are public ‚úÖ

Information quality

  • Are most of the information provided correctly?
  • If not, can the wrong ones get filtered out?

Information correctness

  • Custom information the user can enter can't be "verified"
  • For example the country, city or real name
  • All other information provided by Steam itself are correct

Filter wrong information

  • Real name can be filtered to a certain degree
  • City/Country cannot be filtered at all and can't be trusted therefore

Possible targets and outcomes

Predict gender of user accounts

Pros

  • Interesting because info isn't available directly
  • Enough related data is (likely) available to assume the gender

Cons

  • Limited ground truth
  • Name parsing involved
  • Can fail if not enough (parseable) real names are provided
  • Parse the real name, validate and guess gender based on it
  • Use the "known gender" users as test and learning group
  • Analyze the games played/bought/owned, groups and behaviour

Predict future play time

Pros

  • Prediction algorithms rock!
  • Very valuable information

Cons

  • Can be quite complex, depending on the scope
  • Needs a lot of data for a good prediction
  • Requires time to track the playtimes of a lot of players
  • Unforeseeable events can't be considered
  • Hard to accomplish in the end
  • Likely not what we want to do, just our starting idea

Relation between most-liked genre and amount of money spent

Pros

  • Ground truth is available (steam calculator)
  • Gathering needed data doesn't look too complex

Cons

  • Not all money spent can be tracked
  • Is there a correlation?
  • Steam Calculator for ground truth
  • In-App purchases can't be tracked

Deduce which games a user has based on current games

Pros

  • Outcome can be used as "recommendation system"
  • Data is accessible

Cons

  • High sample size is required
  • Maybe to easy
